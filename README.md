# Room booking project

A simple web application for bookig rooms. Technologies used:<br> <br>
`Python`<br> 
`Django` <br>
`Postgresql` <br>
`HTML` <br>
`Bootstrap`<br>
`Pipenv`<br>
`Git`

# Main app features

* booking rooms
* adding new rooms
* editing existing rooms
* deleting rooms
* displaying one or all rooms data
* room searching
